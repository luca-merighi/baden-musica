# Baden Música 
Landing Page para a minha escola de música utilizando conceitos de **HTML**, **CSS** e **JavaScript**. <br/> 
A página possuí semântica, responsividade e acessibilidade com **CSS** e **JavaScript**.

### Site Construído com:
- **HTML**
- **CSS**
- **JavaScript - DOM**
- **SwiperJS**
- **ScrollReveal**
